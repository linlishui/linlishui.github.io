MySQL 基础用法

概述 MySQL 是常见的关系型数据库，主要用来存储和查询结构化数据。 这篇文章聚焦最基础也最常用的内容，包括连接数据库、建库建表、表结构修改、基本 CRUD，以及主键、索引、事务和常见坑点。 连接数据库 最常见的进入方式是命令行连接： mysql -u root -p 如果是连接指定主机和端口： mysql -h 127.0.0.1 -P 3306 -u root -p 连接成功后，可以先看有哪些数据库： SHOW DATABASES; 退出客户端： exit 建库建表 创建一个测试库： CREATE DATABASE demo_db; 切换到这个库： USE demo_db; 查看当前库里的表： SHOW TABLES; 下面建一张最简单的用户表： CREATE TABLE users ( id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(50) NOT NULL, age INT, city VARCHAR(50), created_at DATETIME DEFAULT CURRENT_TIMESTAMP ); 这里有几个关键点： 1. 字段类型 INT：整数 VARCHAR(50)：最长 50 个字符的字符串 DATETIME：日期时间 2. 约束 PRIMARY KEY：主键，唯一标识一条记录 AUTO_INCREMENT：自增，适合主键 NOT NULL：不能为空 DEFAULT：默认值 可以用下面的命令查看表结构： ...
